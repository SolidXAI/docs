import { LayoutGrid, Users, Play } from 'lucide-react';


Once Testing Hub has been made available in your SolidX application, your product team can begin using it in three simple steps.

<Cards>
  <Card icon={<LayoutGrid />} title="1. Add your product" href="/docs/module-marketplace/testing-hub/applications/register-an-application">
    Give Testing Hub a clear name for the product and decide what your team wants to monitor.
  </Card>
  <Card icon={<Users />} title="2. Choose your team">
    Add the people who should see results and be told when a test needs attention.
  </Card>
  <Card icon={<Play />} title="3. Create and run your first tests" href="/docs/module-marketplace/testing-hub/getting-started/first-test-run">
    Describe the journeys to the MCP AI assistant, then run the test scenarios it creates.
  </Card>
</Cards>

<Callout type="info" title="Before you begin">
  Make sure the product is available for your team to use. If Testing Hub is not yet visible in SolidX, ask your administrator to install it from the [Module Marketplace](/docs/module-marketplace).
</Callout>
