## Tutorial Overview

This tutorial walks you through the SolidX flow from zero to a working Todo app in few minutes.

<Callout type="info" title="What this tutorial is optimized for">

This is a fast, beginner-first walkthrough.

- You start with PostgreSQL in Docker
- You scaffold a fresh SolidX app
- You connect an MCP-capable agent
- You generate a working Todo module with two prompts

</Callout>

By the end, you will have:

- Started PostgreSQL in Docker
- Scaffolded a SolidX app
- Connected an MCP-capable agent to SolidX
- Generated a working `todo-management` module
- Verified the generated CRUD API in Swagger

This tutorial is beginner-friendly. It is meant to show the fastest path to a real generated app, built using SolidX.
