# Tutorial

<!-- Welcome to SolidX admin documentation! SolidX is an enterprise-focused, low-code development platform engineered for today's web applications.  -->

Welcome to the **SolidX Tutorial** !

This section is designed to help you get hands-on with SolidX through guided, real-world examples. Whether you're an admin setting up a school fees portal or a developer building a custom library management system — these tutorials will walk you through the entire process using the SolidX platform’s powerful low-code features.

Explore the tutorials below to learn how to:

- Configure models and relationships
- Build UIs without writing code
- Extend logic with code hooks
- Trigger emails and workflows
- And much more...
