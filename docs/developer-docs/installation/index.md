**What You’ll Do**

Setup a SolidX project with our [create-solid-app](https://www.npmjs.com/package/@solidstarters/create-solid-app) starter template

- Run the below command below to initialize your SolidX project: 
    ```tsx
    npx @solidstarters/create-solid-app
    ```
- Answer some simple prompts to configure your project.
- Voila! Your SolidX project should be up and running.
- Customize SolidX to fit your needs!
