# App Builder

The App Builder is a core module of SOLID that enables you to create and manage your application's data models and user interfaces.

## Key Features

- **Resource Builder**: Define and manage your data models through an intuitive interface
- **Layout Editor**: Create custom views for your data using list, kanban, and form layouts
- **Module Management**: Organize your application into logical modules
- **Field Management**: Configure various field types to structure your data

## Getting Started

The App Builder consists of three main components:

1. [Module Management](./module-management.md) - Organize your application structure
2. [Model Management](./model-management.md) - Define your data models
3. [Field Management](./field-management.md) - Configure data fields and their properties

## Views

The App Builder supports multiple view types to display and interact with your data:

1. [List View](./views/list-view.md) - Display data in a tabular format
2. [Kanban View](./views/kanban-view.md) - Visualize and manage data in a kanban board
3. [Form View](./views/form-view.md) - Create and edit individual records
