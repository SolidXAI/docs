# SOLID Starters - User Documentation

Welcome to SOLID Starters' user documentation! SOLID Starters is an enterprise-focussed, low-code development platform engineered for today's web applications.

## Getting Started

To access the admin panel:
1. Navigate to your SOLID installation URL
2. Log in with your credentials (default credentials: admin@example.com / admin@123)
3. Upon first login, you'll be prompted to change your password

## Core Features

SOLID provides several key features to help you build and manage your applications:

### [App Builder](./app-builder/index.md)
Build and customize your applications effortlessly with our intuitive App Builder. Utilize the resource builder to define models and their fields, and the layout editor to design dynamic interfaces that suit your needs. 

### [Apps](./apps/index.md)
Easily manage your applications and their RESTful API endpoints through a user-friendly interface. The Apps feature allows you to configure, update, and maintain all your application functionalities seamlessly.

### [Media Library](./media-library/index.md)
Comprehensive media library and storage provider options facilitate easy handling and organization of digital assets within applications.

### [Identity and Access Management](./iam/index.md)
Built-in user, role, and permission management features ensure secure access control and compliance with organizational policies.

### [Email & SMS Templates](./notifications/index.md)
Integrated notifications for email, SMS, and push notifications (including Firebase and WhatsApp), allowing customization for user communication.

### [Other Settings](./settings/index.md)
Configure additional platform settings and features.
