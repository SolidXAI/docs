# SOLID User Documentation

Welcome to SOLID's user documentation! SOLID is a powerful enterprise low-code development platform that helps you build and manage applications efficiently.

## Getting Started

To access the admin panel:
1. Navigate to your SOLID installation URL
2. Log in with your credentials (default username: admin)
3. Upon first login, you'll be prompted to change your password

## Core Features

SOLID provides several key features to help you build and manage your applications:

### [App Builder](./app-builder/index.md)
Build and customize your applications using our intuitive resource builder and layout editor.

### [Apps](./apps/index.md)
Manage your applications and their RESTful API endpoints through an intuitive interface.

### [Media Library](./media-library/index.md)
Manage your media files with support for multiple storage providers.

### [Identity and Access Management](./iam/index.md)
Control user access with comprehensive user, role, and permission management.

### [Email & SMS Templates](./notifications/index.md)
Create and manage communication templates for your applications.

### [Other Settings](./settings/index.md)
Configure additional platform settings and features.
